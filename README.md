## 🧩 What is the Riemann Hypothesis?

Formulated by Bernhard Riemann in **1859**, the hypothesis states that all
non-trivial zeros of the Riemann zeta function lie on the **critical line**
where the real part equals exactly **1/2**.

It is one of the **7 Millennium Prize Problems** — worth **$1,000,000** if solved.
It has resisted every attempt for over **160 years**.
