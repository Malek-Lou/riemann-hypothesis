
# The Riemann Hypothesis

## Description
This project is a report written in LaTeX on the Riemann Hypothesis,
exploring the distribution of prime numbers and the zeros of the 
Riemann zeta function. It is structured as the introduction of a 
scientific paper, including context, review of the literature, and 
open directions.


## Files
- `riemann.tex` — main LaTeX source
- `references.bib` — bibliography (BibTeX format)

## Structure
- **Context** : introduction to the zeta function and the conjecture
- **Review of the Literature** : key results from Hardy, Conrey, Odlyzko and Berry & Keating
- **Needs** : open problems and future directions

## Tools
- Written in LaTeX on Overleaf
- Bibliography managed with Zotero
- Version control with Git
